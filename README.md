# 🤖 Rule-Based Chatbot

A simple Rule-Based Chatbot built using **HTML, CSS, and JavaScript**.
The chatbot responds to user messages using predefined rules and pattern matching techniques.

---

## 📌 Project Description

This project demonstrates a basic chatbot system where responses are generated using:

* `if-else` conditions
* Keyword matching
* Simple conversation flow

The chatbot can answer greetings, tell the current date and time, introduce itself, and respond to basic user queries.

---

## 🚀 Features

* Interactive chatbot interface
* Rule-based response system
* Instant replies to user messages
* Displays current date and time
* Enter key support
* Clean and responsive UI
* Beginner-friendly project

---
